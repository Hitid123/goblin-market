# GOBLIN MARKET

Pixel / dark-forest landing page for **Goblin Market** ($GOBLIN) — [@playgoblinmark](https://x.com/playgoblinmark).

Static site — pure HTML / CSS / JS with GSAP + Lenis (via CDN). No build step.

## Sections
Hero (parallax forest, fireflies, glitch logo, hooded goblin) · Marquee · Lore (RPG dialog + counters) · Stalls (hover-reveal market) · Quest Log · Join.

## Run locally
```bash
python -m http.server 5599
# open http://localhost:5599
```

## Deploy
Static — deploys as-is on Vercel / Netlify / GitHub Pages (no config needed).
