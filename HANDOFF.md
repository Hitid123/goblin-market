## ▶️ Запуск локально
Просто откройте `index.html` в браузере.
(Если анимации скролла важны — запустите простой сервер: `python -m http.server` и откройте `http://localhost:8000`.)

## 🚀 Как опубликовать

**Вариант А — Vercel (рекомендую, бесплатно):**
1. Зайдите на https://vercel.com → Add New → Project
2. Импортируйте репозиторий **или** перетащите эту папку
3. Framework Preset = **Other**, Build/Output оставьте пустыми → **Deploy**
4. Получите ссылку `…vercel.app`

**Вариант Б — Netlify (перетаскиванием):**
1. https://app.netlify.com/drop
2. Перетащите всю папку проекта в окно → готово.

**Вариант В — любой хостинг:**
Залейте все файлы в корень сайта по FTP. Это обычная статика.

## 🌐 Свой домен
На Vercel/Netlify: Project → **Domains** → Add → введите домен (напр. `playgoblinmark.xyz`)
и пропишите у регистратора домена записи, которые покажет панель (CNAME/A). Через 5–30 мин заработает.

## ✏️ Где менять контент
Всё текстом в `index.html`:
- **Ссылка на X (Twitter)** — найдите `x.com/playgoblinmark` (несколько мест) и замените на свою.
- **Тексты** — заголовки, описания, лор «Grimtooth», дорожная карта (Quest Log) — обычный текст в `index.html`.
- **Цифры статистики** (6 666 гоблинов и т.д.) — атрибут `data-count` в блоке `lore-stats`.
- **Контракт-адрес / DEX-ссылки** — можно добавить в секцию Join, скажите разработчику куда.
- **Звук** — выключен по умолчанию, включается кнопкой «SOUND» в шапке.
